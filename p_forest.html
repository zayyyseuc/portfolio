Design Goal

A game that does not simply "teach" ecology, but restores a childlike sense of wonder — allowing players to rediscover nature through exploration, storytelling, and experiential learning. Merging the accessibility of textbooks with the immersion of nature, while unlocking the dimension of Micro-Visibility that neither can provide alone.

Research Background

Growing up in an exam-oriented education environment, children develop strong academic skills yet lack basic understanding of the natural world. Today's children know far more about electronic devices than the names of trees, the structure of ecosystems, or how species depend on each other. This project is a response to that disconnection.

Technical Implementation

Built in Unreal Engine 5 using Blueprint visual scripting, the game implements a spatial UI system, multi-perspective camera controls, dynamic NPC behavior, a custom inventory system, and cinematic QTE sequences — all interconnected through event-driven architecture.

Narrative Subtitles
Path-Anchored Spatial Text

Data-driven 3D text deployment using splines for automated placement along the player's path. Creates spline along Gamekey Path, dynamically generates a Box Collision Trigger at the loop node's runtime position, moves the player to that location on trigger.

Ant's Vision
Compound Eye Post-Process

Ant vision: low-resolution mosaic dominated by high-contrast silhouettes and abstract light patterns. Radial blur + Outline shader + Green LUT. Lowers camera Z-axis to ground level. Synchronized physical camera height adjustments via custom post-process materials.

Cinematic QTE
Collaborative Ant Sequence

Three-tier QTE mechanics: 1. Sequencer Event Track triggers input prompts. 2. Director BP bridges manages countdown and response evaluation. 3. BP_QTE_Initialization manages simultaneous input listening, anchors UI prompts, slows game speed during reaction window.

NPC Behavior System

  • Dialogue-driven & UI Interaction Layer — NPC framework with dynamic dialogue and autonomous movement
  • Stores lines in array, uses IncrementInt to cycle through indices. Players get different feedback through repeated interaction
  • Proximity Sensing & AI Behavior — dynamically decides movement logic based on spatial distance between player and NPC
  • Calls NavMesh data to drive NPC movement to the destination with obstacle avoidance

Inventory System

  • Dynamically loads item information from a structured database when player interacts with collection points
  • Type Filtering: Redirects logic to specific array based on the Input Type (Consumables, Key Items, Documents)
  • Complete decoupling of item attributes from core logic (Icon, Desc, Max Stack) from the DataTable
  • Writes newly calculated state to the reduced state matching the input ItemID to ensure precision

Gameplay Captures

Across three chapters, players shift perspectives from human explorer to arboreal squirrel to ground-level ant — each scale unlocking a different ecological layer. The experience density builds from observational curiosity to cooperative urgency.

Chapter 1
Vanished Beast Trails

Players observe how anthropogenic habitat fragmentation disrupts animal movement corridors. Environmental storytelling through absence — where paths once existed, now fragmented by infrastructure.

Chapter 2
Help the Squirrel Gather Nuts

Players inhabit squirrel perspective with Grappling Traversal System. Agile arboreal locomotion — fluid momentum and precision landing. Inventory system tracks collection, foraging routes, and granary organization as ecological resource management.

Chapter 3
Enter the Ant's Field of Vision

Complete sensory shift into micro-scale. Compound eye post-process renders spectral green palette. VOC gradient navigation leads player to dying giant trees — the forest's communication network made visible.

Learning Objectives Embedded

  • Habitat Islands → Canopy Gaps → Deadfall & Logjams: Habitat Fragmentation theory through traversal
  • Conflict Recognition → Barrier Crossing → Resource Clusters: Ecological corridor planning logic
  • Foraging Logic → Seasonal Fluctuation → Altitude-specific species: Ecological Niche theory
  • Pheromone Flow Trails → Thermal Micro-clusters → UV Post-process: Sensory Ecology of insects

Design Opportunity Achieved

  • Safety — explorable virtual forest without physical risk or travel cost
  • Immersion — higher than textbook, comparable to real forest for ecological detail
  • Micro-Visibility — unique: UV pheromone trails, fungal networks, bark micro-texture impossible to see in real forest
  • Engagement — game mechanics sustain attention beyond passive nature documentary format
